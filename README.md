## ⚙️ Installation (Local Setup)

### Backend
1. Clone repository
2. Open project in Eclipse or IntelliJ
3. Configure MySQL database
4. Run Spring Boot Application

### Frontend
```bash
cd student-frontend
npm install
npm start
```
